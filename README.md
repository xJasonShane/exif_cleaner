# EXIF Cleaner

一个用于批量删除图片EXIF信息的工具，支持多种图片格式，提供简洁直观的GUI界面。

## 功能特点

- 📁 **批量处理**：支持单文件和文件夹选择，可批量处理大量图片
- 🎯 **选择性删除**：可选择删除特定的EXIF信息，或删除全部EXIF信息
- 📷 **多格式支持**：支持JPG、JPEG、PNG、WEBP等多种图片格式
- 🔍 **EXIF高亮**：选择图片后，自动高亮显示该图片包含的EXIF标签
- ⏳ **进度显示**：显示处理进度，让您了解处理状态
- 🔄 **手动更新**：可手动检查GitHub仓库的最新版本，提示用户更新
- 🎨 **简洁界面**：直观易用的GUI界面，操作简单
- 📋 **关于界面**：包含软件开源协议、作者信息和GitHub仓库链接
- 💡 **安全处理**：默认创建副本后清理，不修改原始文件
- 🔒 **重复文件过滤**：自动过滤重复文件，避免重复处理
- 📂 **副本保存**：支持自定义副本保存目录，自动添加`_副本`后缀
- 🚫 **无重复添加**：智能检测重复文件，避免重复添加到列表

## 支持的图片格式

- ✅ JPG
- ✅ JPEG
- ✅ PNG
- ✅ WEBP

## 技术栈

- **编程语言**：Python 3.8+
- **GUI框架**：tkinter
- **EXIF处理**：piexif
- **图片处理**：Pillow
- **HTTP请求**：requests
- **开发环境**：虚拟环境

## 安装方法

### 1. 克隆仓库

```bash
git clone <仓库地址>
cd exif_cleaner
```

### 2. 创建虚拟环境

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

## 使用说明

### 运行软件

```bash
python main.py
```

### 基本操作

1. **选择文件**：点击"选择文件"按钮选择单个或多个图片文件
2. **选择文件夹**：点击"选择文件夹"按钮选择包含图片的文件夹
3. **创建副本清理**：点击"创建副本清理"按钮，创建副本后清理EXIF信息
4. **选择性删除**：在EXIF选项中选择要删除的标签，点击"删除所选EXIF"按钮
5. **检查更新**：点击"检查更新"按钮手动检查软件更新
6. **关于软件**：点击"关于"按钮查看软件的开源协议、作者信息和GitHub仓库链接

### EXIF信息查看

- 选择图片后，在"EXIF信息列表"区域会高亮显示该图片包含的EXIF标签
- 未包含的标签将显示为普通样式

### 选择性删除

在"EXIF信息列表"区域可以选择要删除的EXIF标签，支持：

- 全选/取消全选功能
- 高亮显示当前图片包含的EXIF标签

### 副本处理

- **单个文件**：生成带`_副本`后缀的文件名，保存到指定文件夹
- **多个文件**：批量生成带`_副本`后缀的文件名，保存到指定文件夹
- 支持进度显示和取消操作

### EXIF处理方式

1. **创建副本后清理**：不修改原始文件，只处理副本文件
2. **选择性清理**：可以选择要删除的EXIF标签，保留其他标签

## 开发说明

### 模块设计

本项目采用模块化设计，各个功能模块独立，便于后续扩展和维护：

- **exif_processor.py**：处理EXIF信息的核心逻辑
- **file_handler.py**：处理文件选择和拖拽功能
- **version_manager.py**：管理版本信息
- **update_checker.py**：检查更新功能
- **gui.py**：GUI界面设计

### 版本管理

版本信息存储在`config/version.json`文件中，采用语义化版本控制。

### 更新检查

软件提供手动检查更新功能，用户可通过界面上的"检查更新"按钮检查GitHub仓库的最新release版本，如果有更新会提示用户。
